Key Definitions (asm)

Definitions of the KeyCodes required by cpct_isKeyPressed function for assembler programs.  These are 16-bit values that define matrix line in the keyboard layout (Most Significant Byte) and bit to be tested in that matrix line status for the given key (Least Significant byte).  Each matrix line in the keyboard returns a byte containing the status of 8 keys, 1 bit each.
